# markdown-preview

## About The Project

Converting your markdown code to html live in browser.

Parts of the project:

- Input box that converts markdown to html using React Markdown

### Built With

This is built with:

- [ReactJS](https://reactjs.org/)
- [React Markdown](https://github.com/remarkjs/react-markdown)

## Getting Started

To get started you would need to clone this repository

```bash
git clone https://github.com/sk0le/markdown-preview.git

cd markdown-view

npm install
npm start
```
