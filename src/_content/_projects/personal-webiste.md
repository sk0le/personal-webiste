# Personal Website

Personal website built using NextJS and TailwindCSS, blog included.

## About The Project

This is my personal website that I made using NextJS and TailwindCSS, I also made blog that would read from `_content/_blogs` folder and display you my blog posts, also included search functionality.

Parts of the project:

- Home page, this represents me (about, skills, projects).
- Blog page that has latest blogs, search functionality and for every blog post specific page

What I plan to do is to:

- Add some kind of database
- Add comments to blog posts
- Fix error with project description

### Built With

This is built with:

- [NextJS](https://nextjs.org/)
- [React Markdown](https://github.com/remarkjs/react-markdown)
- [TailwindCSS](http://tailwindcss.com/)

## Getting Started

This project has "FREE" license. You can use it however you want.
To get started you would need to clone this repository

```bash
git clone https://github.com/sk0le/personal-webiste.git

cd ./personal-webiste

# using yarn
yarn
yarn dev

# using npm
npm install
npm run dev
```
